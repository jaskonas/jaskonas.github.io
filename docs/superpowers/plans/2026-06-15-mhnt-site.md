# MHNT Site Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build and deploy mhnt.org — a four-page Eleventy site advertising the Machine Has No Tradition seminar, with a rotating SVG etching typelogo, email capture, faculty profiles, and a password-gated alumni Signal QR code.

**Architecture:** Eleventy with Nunjucks templates, a single flat CSS file, and vanilla JS for the typelogo image cycling and alumni password gate. All data (faculty, alumni, image list) lives in `_data/` JSON files so content updates never touch templates. Deployed to Netlify with built-in form handling.

**Tech Stack:** Eleventy 3.x, Nunjucks, vanilla CSS, vanilla JS (Web Crypto API for SHA-256), Netlify Forms, IBM Plex Mono (Google Fonts), Helvetica Neue (system stack)

**Spec:** `docs/superpowers/specs/2026-06-15-mhnt-site-design.md`

## Task 3: SVG typelogo hero

**Files:**
- Create: `src/_includes/hero.njk`
- Modify: `src/assets/js/main.js`
- Modify: `src/_data/images.json`

The hero SVG renders three overlapping layers over a background etching:
1. Grayscale background image (full bleed)
2. Same image, inverted grayscale, clipped to the MHNT letterforms
3. White 30% rect, clipped to the MHNT letterforms

JS picks a random image on load; J/ArrowLeft steps back, K/ArrowRight steps forward.

- [ ] **Step 4: Write image cycling JS in `src/assets/js/main.js`**

```js
(function () {
  // Injected by Eleventy template — see hero.njk usage in index.njk
  const IMAGES = window.MHNT_IMAGES || [];
  if (!IMAGES.length) return;

  let idx = Math.floor(Math.random() * IMAGES.length);

  function setImage(filename) {
    const path = '/assets/images/' + filename;
    const bg = document.getElementById('hero-bg');
    const inv = document.getElementById('hero-inv');
    if (bg) bg.setAttribute('href', path);
    if (inv) inv.setAttribute('href', path);
  }

  document.addEventListener('keydown', function (e) {
    if (!document.getElementById('hero-bg')) return;
    if (e.key === 'j' || e.key === 'ArrowRight') {
      idx = (idx + 1) % IMAGES.length;
      setImage(IMAGES[idx]);
    } else if (e.key === 'k' || e.key === 'ArrowLeft') {
      idx = (idx - 1 + IMAGES.length) % IMAGES.length;
      setImage(IMAGES[idx]);
    }
  });

  setImage(IMAGES[idx]);
})();
```

## Task 10: Alumni password gate (client-side SHA-256)

**Files:**
- Modify: `src/assets/js/main.js`

The password is hashed client-side using the Web Crypto API. The hash is stored as a constant in `main.js`. When the correct password is entered, the QR section is revealed. The password itself never appears in source.

- [ ] **Step 1: Generate the SHA-256 hash of the alumni password**

Jon provides the password. To generate the hash, run this once in a browser console or Node:

```js
// In browser console:
const pw = 'your-password-here';
const encoder = new TextEncoder();
const data = encoder.encode(pw);
const hash = await crypto.subtle.digest('SHA-256', data);
const hex = Array.from(new Uint8Array(hash)).map(b => b.toString(16).padStart(2, '0')).join('');
console.log(hex);
```

Copy the resulting hex string — you'll embed it in the next step.

- [ ] **Step 2: Add password gate JS to `src/assets/js/main.js`**

Add this block after the existing image cycling code. Replace `'PASTE_SHA256_HASH_HERE'` with the hex string from Step 1.

```js
// ── Alumni password gate ──
(function () {
  const HASH = 'PASTE_SHA256_HASH_HERE';
  const form = document.getElementById('signal-form');
  if (!form) return;

  async function sha256(str) {
    const data = new TextEncoder().encode(str);
    const buf = await crypto.subtle.digest('SHA-256', data);
    return Array.from(new Uint8Array(buf))
      .map(b => b.toString(16).padStart(2, '0'))
      .join('');
  }

  form.addEventListener('submit', async function (e) {
    e.preventDefault();
    const input = document.getElementById('signal-password').value;
    const hash = await sha256(input);
    if (hash === HASH) {
      document.getElementById('signal-gate').hidden = true;
      document.getElementById('signal-content').hidden = false;
    } else {
      const err = document.getElementById('signal-error');
      err.hidden = false;
      document.getElementById('signal-password').value = '';
      setTimeout(() => { err.hidden = true; }, 2000);
    }
  });
})();
```
